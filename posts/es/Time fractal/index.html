Fractal Temporal

Este proyecto fue inspirado por code bullet o carykh o un youtuber similar que no puedo recordar. De todas formas la idea no es mia, solo la he implementado en HTML y JS.

¿Alguna vez te has preguntado cómo se vería si dibujaras un reloj analógico (con 2 manijas) y en la punta de cada manija dibujaras el mismo reloj una vez, y otra vez, y otra vez, para siempre? Es un pensamiento bastante específico, pero no puedo ser yo el único. Para la gran mayoria de vosotros que han estado esperando esto toda su vida aqui teneis.

Este es un concepto aparentemente simple, y al principio es un poco aburrido, pero a medida que pasa el tiempo surgen nuevas formas. Puedes abrir la pestaña en tu navegador y dejarla ahí durante un tiempo para que cambie sufiente para ser interesante.
